Project Scope

This is a parts list and description of hardware for a 45-employee business. The cost of the components and the reasoning behind choosing each one can be found below.

Parts List

  • 48-port Switch - $399.95
  • Cat5e Cable (x5) - $469.50
  • Router - $58.00
  • RJ-45 Connectors - $9.99
  • Internet Package (150 Mbps) - $249.95/mo

  • Total = $1187.39 initial cost + $249.95/mo

Switch

This 48-port switch will be able to handle the traffic from all 45 employees, plus a connection to the router and 2 for servers/databases. It is always good to plan for the future, so it might be a good idea to get 2 of these and daisy chain them. Two wouldn't be necessary initially though.

Cables

I assumed that every employee would require 100 ft. of Cat5e cable, probably a little high but it's always good to have extra. So with 45 employees that is 4500 ft. which is why I would opt to get 5000 ft. of cable. I also discovered there are different qualifications for Cat(X) cable which I read about here ( CMR vs CMP). So with that in mind I would also need a bunch of conduit to put the cables in, but I haven't included those in this as I felt it was beyond the scope of this page. There would be a lot of different conduit pieces though that would need to be purchased and it would change depending on the layout of the building.

Router

The router has gigabit ethernet ports which are more then enough for the 150 Mbps maximum bandwidth that will be coming from the ISP. At some point the company may want to get some wireless access points but to start out I felt that this was good.

RJ-45 Connectors

Connectors to go with the Cat5e cable. They would need a crimping tool to go along with these and a cable tester would be a good thing as well, but I just assumed that we the contractors would already have those tools readily available.

Internet Package

With 45 employess and a bandwidth of 150 Mbps, which calculates to 3.33 Mbps to every employee if the traffic is perfectly dispersed. It will never be perfectly dispersed but that should be good enough for everyday office use. They won't all be able to stream Netfilx HD at the same time, but that's okay.
